◆ rvviVersionCheck()

bool_t rvviVersionCheck ( uint32_t  version)

Check the compiled RVVI API version.

Makes sure the ImperasDV version linked with matches the versions defined in this header file. This should be called before any other RVVI API function. If this function returns RVVI_FALSE, no other RVVI API function should be called.

Parameters
versionShould be set to RVVI_API_VERSION.
Returns
RVVI_TRUE if versions matches otherwise RVVI_FALSE.

bool_t rvviRefInit ( const char *  programPath,

const char *  variant,
uint32_t  addressBusWidth 
)

Initialize the DV reference model.

Parameters
programPathFile path of the ELF file to be executed. This parameter can be NULL if required.
vendorVendor string that the reference model will use.
variantVariant string that the reference model will use.
addressBusWidthThe width in bits of the processors address bus. Set to 0 to use the maximum available width.
Returns
RVVI_TRUE if the reference was initialized successfully else RVVI_FALSE.
Note
The reference model will begin execution from the entry point of the provided ELF file but can be overridden by the rvviRefPcSet() function.
